The instructions for this are on the Thunkable Docs page: <span style={{color: '#5162AE',}}> <a target="_blank" rel="noopener noreferrer" href="https://docs.thunkable.com/thunkable-cross-platform/create/components/data/spreadsheet"> https://docs.thunkable.com/thunkable-cross-platform/create/components/data/spreadsheet </a> </span>  </div>
